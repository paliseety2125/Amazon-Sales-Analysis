# Amazon Sales Data Analysis

## Project Summary
Analyzed Amazon sales data to identify revenue trends, top products, and regional performance.

## Tools Used
- Python (Pandas, NumPy)
- SQL (MySQL)
- Power BI (Dashboards)
- Excel (Advanced)

## Key Insights
- Revenue grew 23% in Q4 driven by electronics category
- Top 3 regions contributed 60% of total sales

## Files
- data_cleaning.py — Python script for cleaning raw data
- sales_analysis.sql — SQL queries for trend analysis
- dashboard_screenshot.png — Power BI dashboard preview
